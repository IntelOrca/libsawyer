# libsawyer
Common code shared between [OpenRCT2](https://github.com/OpenRCT2/OpenRCT2) and [OpenLoco](https://github.com/OpenLoco/OpenLoco).

## Build
```
cmake -G Ninja -B bin
cmake --build bin
cmake --install bin --prefix out
```

## Build fsaw
```
cd tools/fsaw
cmake -G Ninja -B bin
cmake --build bin
```
